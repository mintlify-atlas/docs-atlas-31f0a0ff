---
title: "Quickstart: install and extract PII spans in minutes"
description: "Install Privacy Parser and extract your first structured PII spans from text using the regex or hybrid backend in under five minutes."
---

This quickstart walks you through installing Privacy Parser, importing the recommended backend, parsing a string of text, and inspecting the structured spans that come back. By the end you will have a working script that identifies names, emails, phone numbers, addresses, account numbers, and secrets in a single call. If you want to compare all three backends before diving in, see [choosing a backend](/backends/overview). For detailed install options, see the [installation guide](/installation).

<Steps>
  <Step title="Install the library">
    Install Privacy Parser and its `privacy-filter` dependency using `uv`. Both packages must be installed from source.

    ```bash
    uv venv
    uv pip install -e ./privacy-filter
    uv pip install -e ./pii_parser
    ```

    <Note>
      The first time you import `ModelPIIParser` or `HybridPIIParser`, the opf 1.5B checkpoint (~3 GB) is downloaded automatically to `~/.opf/privacy_filter/`. The download only happens once. If you only need the regex backend (`PIIParser`), no model weights are required. See [installation](/installation) for details.
    </Note>
  </Step>

  <Step title="Import the hybrid parser">
    `HybridPIIParser` is the recommended backend. It combines the opf model with span-merging and regex backstops to reach F1 0.929 at roughly 600 ms on CPU.

    ```python
    from pii_parser.hybrid import HybridPIIParser

    parser = HybridPIIParser(device="cpu")
    ```

    Pass `device="cuda"` if you have a GPU available.
  </Step>

  <Step title="Parse a text string">
    Call `parser.parse()` with any string. It returns a `HybridParseResult` whose `spans` attribute holds all detected entities.

    ```python
    result = parser.parse(
        "Hi Quindle Testwick (quindle.testwick@openai.com / +1-415-555-0102), "
        "account 40702810500001234567, 14 Beautiful Ct, Anytown USA, "
        "password Priv4cy-Filt3r-2026."
    )
    ```
  </Step>

  <Step title="Inspect the detected spans">
    Each span in `result.spans` has a `label`, a `text` field, and character offsets (`start`, `end`).

    ```python
    for s in result.spans:
        print(f"{s.label:18}  {s.text}")
    ```

    Expected output:

    ```text
    private_person      Quindle Testwick
    private_email       quindle.testwick@openai.com
    private_phone       +1-415-555-0102
    account_number      40702810500001234567
    private_address     14 Beautiful Ct, Anytown USA
    secret              Priv4cy-Filt3r-2026
    ```

    The spans are sorted by character position. You can also serialize the full result to JSON:

    ```python
    print(result.to_json())
    ```
  </Step>
</Steps>
